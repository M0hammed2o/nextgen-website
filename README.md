# NextGen AI — Marketing Website

Public-facing marketing website for the NextGen AI Platform.

## Features

- Landing page with product overview
- Pricing section
- Features showcase
- Contact / demo request
- Privacy Policy page
- Terms of Service page
- Data Deletion page

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Pages

- `/` — Landing page
- `/privacy` — Privacy Policy
- `/terms` — Terms of Service
- `/data-deletion` — Data Deletion Request
